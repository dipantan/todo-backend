# Task Manager App

## Setup Instructions

### Backend
1. Clone the repo: `git clone your-repo-url`
2. Navigate to `backend/`: `cd backend`
3. Install dependencies: `npm install`
4. Set up `.env` with `MONGO_URI`, `JWT_SECRET`, `PORT`
5. Run: `npm start`
